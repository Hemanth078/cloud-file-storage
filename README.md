# Cloud-Based Secure File Storage System

## Author
G. Hemanth  
Vasavi College of Engineering  

## Description
A cloud-based file storage system using AWS EC2 and S3 with Flask backend.

## Features
- User authentication
- File upload/download/delete
- Secure cloud storage
- Responsive UI

## Technologies
- AWS EC2
- AWS S3
- Python Flask
- SQLite
- Bootstrap

## How to Run
1. Install dependencies
2. Configure AWS IAM role
3. Run: sudo python3 app.py
4. Open browser with server IP
